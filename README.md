# 📊 NixTrade Terminal

**NixTrade Terminal** es una plataforma avanzada de trading y análisis de criptomonedas en tiempo real. El objetivo del proyecto es centralizar el descubrimiento de activos, métricas globales y análisis técnico profundo en una interfaz unificada, rápida y optimizada.

Este proyecto utiliza la **API de CoinGecko** (Versión Gratuita/Demo) como motor principal de datos del ecosistema cripto.

---

## 🚀 Características Principales (Roadmap)

- **🔥 Descubrimiento en Vivo:** Monitoreo en tiempo real de activos en tendencia (*trending*) y categorías impulsadas por el impulso del mercado.
- **🌐 Feed Global de Monedas (Explorador):** Vista completa y detallada de todas las criptomonedas con clasificaciones de mercado, precios actuales y un sistema de **paginación eficiente del lado del servidor (Server-side Pagination)** para un rendimiento óptimo.
- **📈 Análisis Técnico Avanzado:** Integración de gráficos de velas (*candlesticks*) de alta precisión mediante la librería de **TradingView**.
- **⚡ Datos en Tiempo Real:** Análisis profundo de cada activo conectando con la API de WebSockets de CoinGecko para actualizaciones instantáneas de precios y flujos de datos.

---

## 🛠️ Stack Tecnológico

El proyecto está construido sobre un entorno robusto, tipado y optimizado para el rendimiento web:

- **Framework:** [Next.js](https://nextjs.org/) (App Router para renderizado eficiente y Server Actions).
- **Librería UI:** [React](https://react.dev/) & [TypeScript](https://www.typescriptlang.org/) (Tipado estricto para la consistencia de datos de la API).
- **Estilos y Componentes:** [TailwindCSS](https://tailwindcss.com/) & [shadcn/ui](https://ui.shadcn.com/) (Diseño limpio, responsivo y componentes accesibles).
- **Gráficos:** [TradingView Lightweight Charts](https://www.tradingview.com/lightweight-charts/) (Para renderizado de velas de alto rendimiento).
- **Proveedor de Datos:** [CoinGecko API v3](https://www.coingecko.com/es/api) (Plataforma independiente de seguimiento cripto).
